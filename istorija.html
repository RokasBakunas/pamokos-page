Trumpai, HTML istorija

Pirmąją HTML versiją 1993 m. parašė Timas Bernersas-Lee. Nuo tada buvo daug skirtingų HTML versijų. 2000-aisiais plačiausiai naudojama versija buvo HTML 4.01 , kuri oficialiu standartu tapo 1999 m. gruodžio mėn.
Kita versija, XHTML, buvo HTML perrašymas į XML kalbą. XML yra standartinė žymėjimo kalba, naudojama kuriant kitas žymėjimo kalbas. Šiandien naudojama šimtai XML kalbų, įskaitant GML (geografijos žymėjimo kalbą), MathML, MusicML ir RSS (tikrai paprastą sindikaciją). Kadangi kiekviena iš šių kalbų buvo parašyta bendra kalba (XML), jų turinys gali būti lengvai bendrinamas su programomis. Dėl to XML gali būti labai galingas, ir nenuostabu, kad W3C sukurs XML HTML versiją (vėl vadinamą XHTML). XHTML tapo oficialiu standartu 2000 m., o atnaujintas 2002 m. XHTML yra labai panašus į HTML, tačiau turi griežtesnes taisykles. Griežtos taisyklės būtinos visoms XML kalboms, nes be jos programų sąveika būtų neįmanoma.
