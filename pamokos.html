Kas yra HTML?

HTML yra standartinė žymėjimo kalba kuriant tinklalapius.

  • HTML reiškia hiperteksto žymėjimo kalbą
  • HTML yra standartinė žymėjimo kalba kuriant tinklalapius
  • HTML aprašo tinklalapio struktūrą
  • HTML susideda iš daugybės elementų
  • HTML elementai nurodo naršyklei, kaip rodyti turinį
  • HTML elementai žymi turinio dalis, pvz., „tai antraštė“, „tai pastraipa“, „tai nuoroda“ ir kt.
